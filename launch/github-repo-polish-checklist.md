# GitHub Repo Polish Checklist

Use this after the code and docs are ready for public release.

## Manual GitHub settings

- Set the repository description.
- Set repository topics.
- Set the repository homepage only if a real public demo or docs URL exists.
- Enable GitHub Discussions manually if you want community discussion in the repo.

## Release presentation checks

- Verify the README renders correctly on GitHub.
- Verify screenshots in `docs/images/` load on GitHub.
- Verify the `v0.1.0` release page exists and has the intended notes.
- Verify the repository description, topics, and homepage are visible in the repo header.
