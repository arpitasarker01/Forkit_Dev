# Code of Conduct

## Our standard

Be respectful, direct, and constructive.

Expected behavior:

- discuss technical issues in good faith
- critique ideas, not people
- keep claims concrete and evidence-based
- avoid harassment, abuse, and personal attacks

## Scope

This applies to GitHub issues, pull requests, discussions, and community
spaces linked from this repository.

## Reporting

If you experience or observe unacceptable behavior, contact:

- `info@forkit.dev`
